Objetivo

Esse programa web explora sites de notícias e retém informações como conteúdo das notícias, imagens e hiperlinks e os utiliza para preencher um banco de dados que é utilizado posteriormente para predição de categorias usando classificadores baseado em probabilidade bayesiana, aprendizagem supervisionada e PNL.

Modelo N-gram: O texto é dividido num conjunto de n palavras e a classificação dele é calculada baseada na frequência que as sequências apareceram na categoria

As notícias serão categorizadas em 6 classes: Futebol, Política, Economia, Cinema, Celebridades e Cultura.

Projeto

Foi dividido em 3 etapas:

  1. O Web Scrapper que, através de conhecimento prévio da estrutura das páginas a serem lidas, pega informações importante da notícia para classificação por meio da estrutura html do site. Feito em Python
  2. Os classificadores de texto, baseado em unigrama e bigrama, que leem as notícias guardadas em um banco de dados após a etapa anterior para predizer a classificação. Feito em Julia
  3. Exposição de métricas importantes para identificação da qualidade do classificador neste site. Feito em Python e HTML,CSS,JS

Com o aumento da quantidade de palavras em uma sequência, aumenta-se a camada de contexto a ser analisada pelo classificador. Logo, espera-se que o aumento de complexidade melhore o desempenho do classificador

Métricas Escolhidas: Acurácia (Taxa de Acerto), Índice Kappa (O quão melhor o classificador é em relação a predição aleatória), Tempo de Execução.

Database

Os bancos de dados foram divididos em 2 grupos:

  1. Bancos de Dados para Classificador, que podem ser divididos em datasets de treinamento e datasets de teste.
  2. Bancos de Métricas achadas nos testes. São 2: Bancos com dados diários, relativos a cada teste feito; e banco com dados variando o tamanho do banco de treinamento, para análise de evolução de desempenho com o aumento de informação de treinamento.

Nos gráficos diários, o banco de treinamento usado continha 240 notícias distribuídas igualmente entre as classes. Somente para o trigrama que foi usado um menor, com 90 notícias.

Nos gráficos dependentes de vocabulário, foi usado um banco de testes com 40 notícias e o database de treinamento variou o tamanho para posterior análise.
